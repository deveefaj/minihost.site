# minihost.site
minihost.site-এর প্রতিষ্ঠাতা ও মালিক। স্বশিক্ষিত ওয়েব ডেভেলপার, ছোট ব্যবসা ও ব্যক্তিগত প্রকল্পের জন্য সাশ্রয়ী হোস্টিং সমাধান দেন। দ্রুত, নিরাপদ ও নির্ভরযোগ্য সেবা নিশ্চিত করেন। অনলাইন প্রাইভেসি ও ডেটা সুরক্ষায় আগ্রহী। সামাজিক মাধ্যমে কম সক্রিয়, কাজ দিয়েই নিজেকে পরিচিত করতে চান। সাধারণ মানুষকে সহজে ওয়েবসাইট তৈরির সুযোগ দেওয়াই তার লক্ষ্য।
